# 📧 Gmail自動スプレッドシート転記ツール

指定した条件に合うメールをGmailから自動で検索し、Googleスプレッドシートへ一覧化するツールです。

## ⚙️ 導入・設定方法

### 1. スプレッドシートの準備
1. 新規にGoogleスプレッドシートを作成します。

### 2. プログラム（GAS）の貼り付け
1. スプレッドシートのメニューから **「拡張機能」 ＞ 「Apps Script」** をクリックします。
2. 画面に最初から入っている `function myFunction() {}` をすべて消去します。
3. 同フォルダにある `code.js` の内容をコピーし、そのまま貼り付けます。
4. プロジェクトを保存（フロッピーディスクのアイコンをクリック）します。

### 3. 動作テストと権限承認
1. 対象となるメール（例：件名に `【問い合わせ】` が含まれるメール）をご自身のGmail宛てにテスト送信します。
2. Apps Script画面上部の **「実行」** ボタンをクリックします。
3. 初回実行時のみ、アクセス権限の承認を求めるポップアップが表示されます。画面の指示に従って承認を行ってください。
    *   *「承認が必要です」 ＞ 「権限を確認」 ＞ Googleアカウントを選択 ＞ 詳細をクリック ＞ importGmailToSpreadsheet（安全ではないページ）に移動 ＞ 許可* の順に進めます。
4. スプレッドシートに戻り、メール情報が自動で転記されていることを確認します。

### 4. トリガー（定期自動実行）の設定
1. Apps Script画面の左側メニューにある **「トリガー（目覚まし時計のアイコン）」** をクリックします。
2. 右下の **「トリガーを追加」** ボタンをクリックします。
3. 以下の条件で設定し、保存します。
    *   実行する関数を選択：`importGmailToSpreadsheet`
    *   イベントのソースを選択：`時間主導型`
    *   時間ベースのトリガーのタイプを選択：`時間周期タイマー`
    *   時間の間隔を選択：`1時間ごと` (または任意の間隔)
