# 📊 Yahoo!ニュース自動収集スクレイピングツール

Yahoo!ニュースの主要トピックスから「タイトル」と「詳細URL」をスクレイピングし、画面にリアルタイムで出力しつつ、CSVファイルに自動保存するPythonスクリプトです。

本ツールは他のプロジェクトと競合を起こさないよう、仮想環境（venv）を使用してクリーンに構築・動作させる設計になっています。

## ⚙️ 導入・実行方法

Windows環境を前提として説明します。

### 1. プロジェクトフォルダへの移動
コマンドプロンプトを開き、本ツールのフォルダ（`02_yahoo_news_scraper`）に移動します。

```bash
cd 02_yahoo_news_scraper
```

### 2. 仮想環境の作成とアクティベート
1. 以下のコマンドを実行して仮想環境を作成します。
    ```bash
    python -m venv .venv
    ```
2. 作成した仮想環境を有効化（アクティベート）します。
    ```bash
    .venv\Scripts\activate
    ```
    *   ※成功するとコマンドプロンプトの左端に `(.venv)` と表示されます。

### 3. ライブラリのインストール
仮想環境内に必要なライブラリを一括でインストールします。
```bash
pip install -r requirements.txt
```

### 4. スクリプトの実行
```bash
python scrape.py
```

*   実行後、コマンドプロンプト上にニュースが表示され、同フォルダ内に `yahoo_news.csv` が保存されます。
*   作業を終了する場合は `deactivate` コマンドで仮想環境から抜け出します。
