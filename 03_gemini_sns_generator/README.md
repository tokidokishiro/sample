# 🧠 Gemini API連携 SNS投稿文自動作成ツール

Googleスプレッドシートに入力された「キーワード・テーマ」と「投稿するSNS種類」をトリガーにして、GoogleのAIモデル「Gemini（gemini-1.5-flash）」を用いて魅力的なSNS投稿文（絵文字、ハッシュタグ付き）を自動生成するGASツールです。

## ⚙️ 導入・設定方法

### 1. スプレッドシートの準備
1. 同フォルダにある `sns_ai_template.xlsx` をGoogleドライブにアップロードし、Googleスプレッドシートとして開きます。

### 2. GeminiのAPIキーを取得する（完全無料）
1. [Google AI Studio](https://aistudio.google.com/) にアクセスし、Googleアカウントでログインします。
2. **「Get API key」** ボタンをクリックします。
3. **「Create API key」** をクリックして、作成されたAPIキーをコピーします。

### 3. プログラム（GAS）の貼り付け
1. スプレッドシートのメニューから **「拡張機能」 ＞ 「Apps Script」** をクリックします。
2. 画面のコードエディタに最初から入っているコードをすべて消去します。
3. 同フォルダにある `code.js` の内容をコピーし、そのまま貼り付けて保存します。

### 4. APIキーの安全な登録（スクリプトプロパティ）
APIキーをプログラム内に直接書かず、安全に管理するGoogle推奨の設定方法です。
1. Apps Script画面の左側メニューにある **「プロジェクトの設定（歯車のアイコン）」** をクリックします。
2. 画面を一番下までスクロールし、**「スクリプト プロパティを追加」** をクリックします。
3. 以下の通り設定し、**「スクリプト プロパティを保存」** をクリックします。
    *   プロパティ名：`GEMINI_API_KEY`
    *   値：`（手順2でコピーしたAPIキー）`

### 5. 実行と承認
1. 左側メニューから **「エディタ（フォルダアイコン）」** に戻ります。
2. 上部の実行する関数名が `generateSNSPosts` になっていることを確認し、**「実行」** をクリックします。
3. 初回実行時のみ、アクセス権限の承認画面が出ますので、画面の指示に従って承認（許可）を行ってください。
4. スプレッドシートに戻ると、自動で文章生成が開始され、C列に結果が書き込まれます。
