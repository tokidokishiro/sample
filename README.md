# 🚀 クラウドワークス・ポートフォリオ用自動化ツール集

本リポジトリは、Google Apps Script (GAS) および Python を用いた業務自動化・効率化ツールのサンプル集です。
実務で即座に導入できるクオリティのモックアップを収録しています。

---

## 🛠 収録ツール一覧

### 1. 📧 [Gmail自動スプレッドシート転記ツール](./01_gmail_to_spreadsheet)
*   **使用技術**: Google Apps Script (GAS)
*   **概要**: 特定の条件（送信元、件名など）に一致するメールを検知し、Googleスプレッドシートへ「受信日時」「送信元」「件名」「本文抜粋」「メールリンク」を自動で書き出します。
*   **特徴**: 二重取り込みを防止するためのラベル自動付与機能を備えています。お問合せ管理の自動化に最適です。

### 2. 📊 [Yahoo!ニュース自動収集スクレイピングツール](./02_yahoo_news_scraper)
*   **使用技術**: Python (Requests, BeautifulSoup)
*   **概要**: Yahoo!ニュースの主要トピックスから「タイトル」と「記事詳細URL」を自動でスクレイピングし、コンソール（黒い画面）に表示しつつ、CSVファイルとしてローカルに保存します。
*   **特徴**: 他のPython環境と競合しないよう、仮想環境（venv）を用いた安全でクリーンな構築マニュアルを完備しています。

### 3. 🧠 [Gemini API連携 SNS投稿文自動作成ツール](./03_gemini_sns_generator)
*   **使用技術**: Google Apps Script (GAS), Google Gemini API
*   **概要**: スプレッドシートに入力された「キーワード」と「投稿するSNSの種類」を読み取り、Googleの最新AI「Gemini（gemini-1.5-flash）」を用いて魅力的なSNS投稿文（ハッシュタグ付き）を自動生成します。
*   **特徴**:
    *   APIキーなどの機密情報を安全に管理するスクリプトプロパティ設計。
    *   無料枠内で動作可能なため、API使用料完全無料での運用が可能です。
